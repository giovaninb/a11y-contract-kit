# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 1.1.x   | Yes                |
| 1.0.x   | Best effort        |
| < 1.0   | No                 |

## Reporting a Vulnerability

Please report security issues privately by opening a GitHub Security Advisory or contacting the maintainers directly.

Do not disclose security vulnerabilities in public issues.

We aim to acknowledge reports within 7 days and provide a remediation timeline when possible.
